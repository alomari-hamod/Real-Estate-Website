# RealEstateX - Full-Stack Real Estate Platform

A native full-stack real estate web application built using PHP and JavaScript. The platform allows clients to browse, filter, and purchase properties, while providing an administrative control panel for the property owner to manage all data dynamically.

---

## 🚀 Features

### 1. User Interfaces & Client Experience
* **Main Homepage:** Dynamically showcases the most visited and top-selling properties pulled directly from the database.
* **Shopping Marketplace:** A clean marketplace where users can browse, search, and view all available apartments or houses.
* **Detailed Property View:** Each property features specific descriptions, exact room counts, detailed layouts, and its precise location on an interactive map.
* **Cart & Checkout Gateway:** A dedicated interface to calculate total order prices and simulate secure payment processing.
* **User Authentication:** A secure PHP-driven Login and Sign-Up gateway separating regular clients from the site owner.

### 2. Admin & Owner Dashboard
* **Full Control (CRUD Operations):** The owner can add new apartments, update prices/descriptions, change map coordinates, or delete listings directly from the UI.
* **Live Updates:** Any change made in the dashboard instantly reflects across the online database to the client-facing website.

---

## 🛠️ Tech Stack

* **Frontend:** Native HTML5, CSS3 (Responsive Design), and JavaScript (ES6+) for dynamic UI interactions.
* **Backend:** PHP (Server-side scripting handling business logic, sessions, and routing).
* **Database & Environment:** MySQL database managed via XAMPP Server (Apache & phpMyAdmin) connected to an online data host.
